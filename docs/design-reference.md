## 📱 响应式设计

### 断点设置
- **手机**: < 768px
- **平板**: 768px - 1024px  
- **桌面**: > 1024px

### 移动端适配
- 侧边栏折叠为抽屉菜单
- 帖子列表采用卡片堆叠
- 触摸友好的按钮大小
- 手势操作支持
